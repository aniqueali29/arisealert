- 🌈 **Customizable Colors** - Match your brand or theme.
- ⚡ **Zero Dependencies** - Lightweight and fast.

## 🎯 Basic Usage

\`\`\`javascript
// Simple alert
SoloAlert.fire({
    title: 'Hello!',
    message: 'Welcome to SoloAlert!',
    icon: '👋'
});

// Quick methods
SoloAlert.success('Success!', 'Task completed');
SoloAlert.error('Error!', 'Something went wrong');
\`\`\`
