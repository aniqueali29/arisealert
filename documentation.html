Theme Control

Theming

Control global theme and see individual theme overrides

Current Global Theme: Dark

The first button follows your global theme. The other buttons override it temporarily.

// Method 1: Use current global theme
AriseAlert.success('Success!', 'Uses whatever theme is currently set');

// Method 2: Force a specific theme (overrides global)
AriseAlert.fire({
    title: 'Custom Theme Alert',
    message: 'This ignores global theme',
    type: 'info',
    theme: 'dark' // Forces dark even if global is light
});

// Method 3: Change global theme for all future alerts
AriseAlert.setGlobalTheme('light'); // All alerts now use light theme

Confirmation Dialog

Interactive

User confirmation with callback

AriseAlert.fire({
  title: 'ENTER DUNGEON?',
  message: 'High-risk S-Rank dungeon',
  icon: 'shield',
  showCancelButton: true,
  confirmButtonText: 'Enter',
  cancelButtonText: 'Retreat'
}).then((result) => {
  if (result.confirmed) {
    AriseAlert.success('DUNGEON ENTERED!', 'Prepare for battle!');
  }
});

Multi-Step Alert

Advanced

Chained alerts for multi-step processes

AriseAlert.fire({
  title: 'STEP 1 OF 3',
  message: 'Choose weapon type:',
  input: 'select',
  options: {'sword': 'Sword', 'dagger': 'Dagger'},
  confirmButtonText: 'Next'
}).then(result => {
  if (result.confirmed) {
    AriseAlert.fire({
      title: 'STEP 2 OF 3',
      message: 'Set power level:',
      input: 'number',
      confirmButtonText: 'Next'
    }).then(result2 => {
      if (result2.confirmed) {
        AriseAlert.fire({
          title: 'STEP 3 OF 3',
          message: 'Enter motto:',
          input: 'text',
          confirmButtonText: 'Complete'
        });
      }
    });
  }
});

🔧 Main Methods


AriseAlert.fire(options) → Promise

Main method to display alerts with full customization options.

AriseAlert.success(title, message) → Promise

Quick success alert with green theme.

AriseAlert.error(title, message) → Promise

Quick error alert with red theme and shake animation.

AriseAlert.warning(title, message) → Promise

Quick warning alert with orange theme and pulse effect.

AriseAlert.info(title, message) → Promise

Quick info alert with blue theme.

AriseAlert.toast(options) → Promise

Non-blocking toast notification.

AriseAlert.setColors(colors) → void

Set custom color scheme for all alerts.

🎨 Theme Methods


AriseAlert.setGlobalTheme(theme) → void

Set global theme for all alerts: 'dark' or 'light'

theme: 'light' | 'dark' option

Individual alert theme override in options object

⚙️ Configuration Options


title
string
Alert title text
message
string
Alert message (supports HTML)
icon
string
Icon or emoji to display
type
string
'default' | 'success' | 'error' | 'warning'
theme
string
'dark' | 'light' - Individual theme override
showCancelButton
boolean
Show cancel button
timer
number
Auto-close timer in milliseconds
input
string
'text' | 'email' | 'password' | 'textarea' | 'select'

⚡ Performance Tips

🎯 Minimize DOM operations: Use single fire() calls instead of multiple quick methods
🔄 Reuse instances: Set colors once globally rather than per alert
📱 Mobile optimization: Use shorter messages and larger touch targets
♿ Accessibility: Always provide meaningful titles and ARIA labels
